# *Spotify Top 50 Analytics Dashboard*

A structured Power BI solution designed to analyze Spotify’s Top 50 dataset and deliver clear, actionable insights for data, marketing, and content strategy teams.

---

## *Overview*

This project converts raw Spotify ranking data into a refined analytical dashboard. It provides visibility into key performance indicators, artist contribution patterns, song characteristics, and historical trends. The dashboard supports informed decision-making for playlist curation, audience targeting, and content planning.

## *Business Challenges Addressed*

* Limited visibility in raw ranking data
* No structured KPIs for performance tracking
* No comparative analysis of explicit vs non-explicit content
* Lack of trend identification across time periods
* No integrated view connecting artists, songs, and album types

This dashboard establishes a unified analytical framework to overcome these gaps.

---

## *Business Value*

* Supports data-driven playlist and marketing decisions
* Identifies high-impact songs and artists
* Highlights emerging trends and audience preferences
* Enhances planning for promotional and engagement strategies
